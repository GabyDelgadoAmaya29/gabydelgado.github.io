    Formularios

    Ejemplos de formularios

    Se utiliza un formulario HTML para recopilar la entrada del usuario. La entrada del usuario se envía con mayor frecuencia a un servidor para su procesamiento.

    Ejemplo






    En código html seria asi

    <form action="/action_page.php"> <label for="fname">Nombres:</label>
    <input type="text" id="fname" name="fname" value="Miguel" />
    <label for="lname">Apellidos:</label>
    <input type="text" id="lname" name="lname" value="Rodríguez" />

    <input type="submit" value="Enviar" /> </form>

    Si hace clic en el botón "Enviar", los datos del formulario se enviarán a una página llamada "/action_page.php".

    Campos de texto

    define <input type="text"> un campo de entrada de una sola línea para la entrada de texto.

    Ejemplo




    El elemento <label> </label>

    La etiqueta <label> define una etiqueta para muchos elementos de formulario. El elemento <label> es útil para los usuarios de lectores de pantalla, porque el lector de pantalla leerá en voz alta la etiqueta cuando el usuario se centre en el elemento de entrada. El elemento <label> elemento también ayuda a los usuarios que tienen dificultades para hacer clic en regiones muy pequeñas (como botones de radio o casillas de verificación), porque cuando el usuario hace clic en el texto dentro del elemento <label>, alterna el botón de radio/casilla de verificación. El atributo for de la etiqueta <label> debe ser igual al idatributo del elemento <input> para unirlos.

    Botones de radio

    define <input type="radio"> como un botón de opción. Los botones de opción le permiten al usuario seleccionar UNA de un número limitado de opciones.
