    Párrafos

    Párrafos HTML

    El elemento HTML define un párrafo. Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.

    Pantalla HTML

    No puede estar seguro de cómo se mostrará HTML. Las pantallas grandes o pequeñas y las ventanas redimensionadas crearán resultados diferentes. Con HTML, no puede cambiar la visualización agregando espacios adicionales o líneas adicionales en su código HTML.

    Reglas horizontales HTML

    La etiqueta "hr" define una ruptura temática en una página HTML y, con mayor frecuencia, se muestra como una regla horizontal.

    Saltos de línea HTML

    El elemento HTML "br" define un salto de línea.

    Titulos

    Títulos en HTML

    Los encabezados HTML se definen con las etiquetas "h1" a "h6". "h1"define el encabezado más importante. "h6" define el encabezado menos importante.

    Ejemplos

    Título 1

    Título 2

    Título 3

    Título 4

    Título 5
    Título 6

    Los encabezados son importantes

    Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web. Los usuarios a menudo hojean una página por sus títulos. Es importante utilizar encabezados para mostrar la estructura del documento. "h1" los encabezados deben usarse para los encabezados principales, seguidos de "h2" los encabezados, luego los menos importantes "h3", y así sucesivamente.

    Formato

    Comandos

    El comando <i> sirve para poner itálica cualquier texto, ya sea párrafo o títulos, incluso palabras en específico de un párrafo.

    El comando <b> sirve para poner en negritas una palabra en específico de cualquier texto, sea párrafo o un título.

    El comando <strong> sirve para poner o mostrar un texto importante de un párrafo o título.

    El comando <em> sirve para enfatizar un texto de un párrafo o título.

    El comando <mark> sirve para poner o mostrar un texto marcado de un párrafo o título.

    El comando <small> sirve para poner o mostrar un texto más pequeño de un párrafo o título.

    El comando <del> sirve para poner o mostrar un texto eliminado de un párrafo o título.

    El comando <ins> sirve para poner o mostrar un texto insertado de un párrafo o título.

    El comando <sub> sirve para poner o mostrar un texto en subíndice de un título.

    El comando <sup> sirve para poner o mostrar un texto en superíndice de un título.
